# Mik4gmon

Монитор LTE сигнала для модема Fibocom L850-GL через MikroTik RouterOS.

## Возможности

- Мониторинг RSRP, SINR, RSRQ, RSSI
- Определение активного Band и агрегации несущих (CA)
- Отображение оператора, Cell ID, типа сети
- График скорости приёма/передачи
- История вышек (автоматическая запись при смене Cell ID)
- Сканирование окружающих сот
- Алерт при слабом сигнале (RSRP)
- Управление бендами LTE
- Системный трей (Windows)
- Тёмная тема
- Русский/Английский интерфейс

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python mikrotik_monitor.py --password ПАРОЛЬ
```

### Аргументы

| Аргумент | Описание |
|----------|----------|
| `--host` | IP роутера (по умолч. 192.168.88.1) |
| `--port` | Порт API (по умолч. 8728) |
| `--password` | Пароль API |
| `--interval` | Интервал обновления в секундах |
| `-v` | Подробный вывод |
| `--dark` | Тёмная тема |
| `--lang` | Язык (ru/en) |

### Сборка

**Windows:**
```bash
build_win.bat
```

## Требования

- Python 3.9+
- MikroTik RouterOS с включённым API (порт 8728)
- Модем Fibocom L850-GL (интерфейс LTE1)
